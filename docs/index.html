See it in action

Type artist or song names in Suno's style field and click Create. SunoMate instantly detects and replaces each name with an accurate, detailed musical style description. Multiple names? It handles them all at once.

Add Artist - Expand your database

Don't see your artist? Click the "Add Artist/Song" button that appears in Suno. Type the style description manually or use AI to generate it. Your additions are saved locally and work instantly next time.

AI Generator - AI-powered prompts

Use the extension popup to convert any artist name or custom request into a detailed Suno prompt using AI. Supports Google Gemini, Claude, GPT, and Grok. Your API key stays private in your browser.
